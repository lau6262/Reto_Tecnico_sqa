Automatización con Screenplay Pattern
Proyecto de automatización de pruebas E2E del datepicker de jQuery UI usando Serenity BDD,
Cucumber y Selenium WebDriver.

¿Qué es este proyecto?
Este proyecto automatiza pruebas del componente datepicker 
usando el patrón Screenplay, que organiza el código en:
Tasks: Acciones que realiza el usuario (Ej: Seleccionar una fecha)
Questions: Verificaciones (Ej: ¿La fecha está en formato correcto?)
UI Elements: Elementos de la página web

Tecnologías
Java 17
Gradle - Gestión de dependencias
Serenity BDD - Framework de automatización
Cucumber - Escenarios en lenguaje natural
Selenium - Control del navegador
